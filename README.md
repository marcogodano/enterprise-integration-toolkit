# Enterprise Integration Toolkit

A collection of reusable .NET libraries and components designed to simplify enterprise integration projects.

## Planned modules

- FTP / SFTP
- REST API Client
- SOAP Client
- XML & JSON Processing
- CSV & Excel Import/Export
- File Watcher
- Retry Policies
- Logging
- Scheduling
- Email Services
- Authentication (OAuth2, JWT)
- SAP Integration Helpers
- Database Utilities

## Technologies

- .NET 9
- ASP.NET Core
- Entity Framework Core
- Serilog
- Polly
